# automation-portal

`automation-portal` 是自动化测试平台的 React 门户层。

当前只创建最小骨架与配置，占位为主，不预填完整业务代码。

## 当前前端骨架

- `index.html`
- `package.json`
- `tsconfig.json`
- `tsconfig.node.json`
- `vite.config.ts`
- `src/main.tsx`
- `src/App.tsx`

## 预留目录

- `src/pages/`
- `src/components/`
- `src/layouts/`
- `src/api/`
- `src/types/`
- `public/`

## 本地准备

建议本机先准备：

- Node.js 20 LTS
- npm

安装完后在 `automation-portal` 目录执行：

```powershell
npm install
npm run dev
```

## 说明

- 旧的 Python 服务骨架已按当前方案移除。
- 当前前端入口只保留最小占位内容，方便你后续自己逐步实现页面和接口调用。
